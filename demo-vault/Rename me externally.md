# Rename me externally

This is the note the [[01 External renames]] demo renames. Do **not** rename it inside Obsidian - the whole point is to rename it from *outside* the app (your OS file manager, a terminal, or `git`) while Obsidian stays open.

Two notes link here - [[References/Link A]] and [[References/Link B]]. After you rename or move this file on disk, External Rename Handler recognizes it as a rename and Obsidian updates those links to follow it.

Try it with the steps in [[01 External renames]] or [[02 Moving and folders]].
